# Cole Codes (Portfolio)

## Overview

A single-page personal portfolio for Cole Reilly, built with the Next.js App Router. Highlights selected real-world client work and projects on one route with smooth in-page navigation.

</p>

---

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Vercel

---

## Run Locally

```
git clone https://github.com/colecodesdev/cole-codes.git
cd cole-codes
npm install
npm run dev
```
